# AI Interview Simulator

AI-powered interview simulator that generates personalised interview
questions and provides structured feedback on your responses.

------------------------------------------------------------------------

## Overview

This project is a full-stack AI application that:

-   Generates tailored interview questions based on a CV and job
    description
-   Simulates timed or untimed interview sessions
-   Transcribes spoken responses
-   Evaluates answers using structured NLP-based scoring
-   Provides clear, category-based feedback

The system is designed to simulate realistic interview conditions and provide clear, actionable feedback to help users improve their performance.

------------------------------------------------------------------------

## Features (Planned / In Progress)

-   CV & Job Description analysis for question generation
-   Behavioural and technical question generation
-   Timed interview simulation
-   Audio recording and transcription
-   Structured scoring system:
    -   Question alignment
    -   STAR method completeness
    -   Specificity and clarity
    -   Time efficiency (timed mode)
-   Feedback dashboard with score breakdown
